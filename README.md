LightSTOMP
==========

A lightweight [STOMP](https://stomp.github.io/index.html) client for Java.

Features
--------

* Provides a STOMP over WebSocket client for Java.
* Currently, it is a minimal subset of the STOMP protocol, supporting frames:
  * CONNECT / CONNECTED
  * SUBSCRIBE
  * MESSAGE
  * SEND
